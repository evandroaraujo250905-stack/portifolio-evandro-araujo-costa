# 🌍 Sistema Inteligente de Análise de Microclima e Simulação de Fuga

## 📝 Descrição do Projeto
Este projeto reúne dois módulos distintos que simulam cenários reais utilizando lógica computacional e tomada de decisão automatizada.

O primeiro módulo realiza uma **análise de microclima urbano**, avaliando fatores como temperatura, qualidade do ar e umidade para gerar uma nota final de conforto ambiental em diferentes regiões.

O segundo módulo consiste em um **simulador de evacuação**, onde um agente percorre diferentes ambientes enfrentando obstáculos, gerenciando recursos e tomando decisões até alcançar uma saída segura ou esgotar sua energia.

O projeto foi desenvolvido com foco em **lógica algorítmica, estrutura de decisão e simulação de cenários**, podendo ser aplicado em estudos de cidades inteligentes, segurança e análise ambiental.

---

## 🌡️ Módulo 1: Análise de Microclima
Este módulo avalia diferentes locais com base em três variáveis principais:

- Temperatura
- Qualidade do ar
- Umidade

Cada variável recebe uma pontuação baseada em faixas ideais, e ao final é calculada uma **nota média geral**, representando o nível de conforto ambiental do local.

### 🎯 Objetivo
Identificar quais regiões apresentam melhores condições climáticas para bem-estar humano.

---

## 🚪 Módulo 2: Simulação de Saída de Fuga
Este módulo simula um agente tentando escapar de um ambiente com múltiplos obstáculos.

Durante o percurso, o sistema considera:
- Estados dos locais (livre, trancado, necessidade de voltas)
- Coleta de itens (como chave)
- Consumo de energia
- Condições de sucesso ou falha

### 🎯 Objetivo
Simular tomadas de decisão em tempo real com base em recursos limitados e condições variáveis.
