# 🛒 Sistema de Cálculo de Compras e Troco

## 📝 Descrição do Projeto

Este projeto consiste em um sistema desenvolvido em pseudocódigo para
realizar o cálculo de compras e gerenciamento de troco em um processo de
pagamento.

O sistema foi estruturado com base em quatro funções principais,
representadas por fluxogramas: - **Total da Compra** - **Calcular Valor
Pago** - **Troco do Cliente** - **Cálculo das Notas de Troco**

O objetivo principal é automatizar o processo de pagamento, garantindo
precisão no cálculo dos valores e eficiência na devolução do troco ao
cliente.

O desenvolvimento foi guiado por fluxogramas e pseudocódigo, permitindo
uma organização clara da lógica e facilitando a futura implementação em
linguagem de programação.

## 🚀 Tecnologias Utilizadas

-   **Linguagem:** Pseudocódigo
-   **Ferramentas:** Lógica de Programação
-   **Metodologia:** Fluxograma + Modularização de Funções

## 📊 Resultados e Aprendizados

O projeto proporcionou melhor entendimento sobre divisão de problemas em
funções e organização lógica.

-   **Modularização:** Separação do sistema em funções independentes.
-   **Precisão de Cálculo:** Garantia de resultados corretos em
    operações financeiras.
-   **Raciocínio Lógico:** Desenvolvimento de algoritmos estruturados.
-   **Aplicação Prática:** Simulação de um sistema real de caixa.

## 🔧 Como Executar

1.  Analise os fluxogramas de cada função.
2.  Siga a lógica descrita no pseudocódigo.
3.  Implemente em uma linguagem de programação (opcional).
